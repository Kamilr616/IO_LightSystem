# External references

Third-party standards, data sheets, user manuals and schematics are not distributed with the release. Their publishers retain all rights; use the official sources below.

## Road-lighting standards

These documents were historical inspiration only and are not implementation requirements:

- [PKN-CEN/TR 13201-1:2016-02 — Guidelines on selection of lighting classes](https://sklep.pkn.pl/pkn-cen-tr-13201-1-2016-02e.html)
- [PN-EN 13201-2:2016-03 — Performance requirements](https://sklep.pkn.pl/normy/pn-en-13201-2-2016-03e.html)

The PKN copies previously present in development history carried an institution-specific license and an explicit prohibition on reproduction without written permission. They are intentionally absent from release `v1.1.0`.
