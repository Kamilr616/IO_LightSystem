# Odnośniki zewnętrzne

Normy, karty katalogowe, instrukcje i schematy podmiotów trzecich nie są rozpowszechniane z wydaniem. Prawa zachowują ich wydawcy; korzystaj z poniższych oficjalnych źródeł.

## Normy oświetlenia drogowego

Dokumenty były wyłącznie historyczną inspiracją i nie stanowią wymagań implementacyjnych:

- [PKN-CEN/TR 13201-1:2016-02 — Wytyczne dotyczące wyboru klas oświetlenia](https://sklep.pkn.pl/pkn-cen-tr-13201-1-2016-02e.html)
- [PN-EN 13201-2:2016-03 — Wymagania eksploatacyjne](https://sklep.pkn.pl/normy/pn-en-13201-2-2016-03e.html)

Kopie PKN obecne wcześniej w historii rozwojowej miały licencję przypisaną do konkretnej instytucji i wyraźny zakaz powielania bez pisemnej zgody. Dlatego celowo nie występują w wydaniu `v1.1.0`.
